# Overview

Application creates table with statistics about selected video project.

Application key points:
- In-depth statistics for labeling jobs
- Review labeling jobs done by annotators
- Сheck the speed and quality of annotations
- Analyze all labeling jobs in one application


# How to Run

### 1. Add [Action Recognition Stats Tool](https://ecosystem.supervise.ly/apps/action-recognition-stats) to your team from Ecosystem.
